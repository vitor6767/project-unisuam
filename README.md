

# Projeto Acadêmico — UNISUAM

Projeto acadêmico desenvolvido em grupo como parte das atividades do curso de
Análise e Desenvolvimento de Sistemas, com o objetivo de criar um site para
venda de produtos de cabelo.

O foco do projeto foi aplicar conceitos de desenvolvimento Front-end,
utilizando HTML, CSS e ajustes básicos em JavaScript.

## Contexto do projeto

O código inicial do projeto foi gerado com auxílio de ferramentas automáticas,
apresentando problemas de estrutura, organização e layout.

Minha atuação principal foi analisar o código existente, identificar erros
e realizar ajustes manuais para melhorar a estrutura do HTML, o CSS
e a responsividade do site.

## Minhas responsabilidades

- Correção da estrutura HTML
- Ajustes e organização do CSS
- Melhoria do layout e espaçamento
- Adaptação para diferentes tamanhos de tela (responsividade)
- Ajustes básicos em funcionalidades JavaScript existentes

## O que aprendi com o projeto

- Leitura e entendimento de código existente
- Identificação e correção de problemas estruturais
- Organização de código Front-end
- Melhoria visual e adaptação de layout
- Trabalho em projeto acadêmico em grupo

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ajustes básicos)

## Observações

Este projeto foi desenvolvido para fins acadêmicos e aprendizado,
representando meu nível técnico no momento de sua criação.
